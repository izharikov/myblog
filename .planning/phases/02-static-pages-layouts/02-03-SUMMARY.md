## What Was Built

### src/pages/index.astro
- Fetches all blog posts via `getCollection('blog')`
- Sorts by date descending using `parseDate` from date.ts
- Slices first 3 posts and maps to BlogGrid-compatible shape
- Renders HeroSection and BlogGrid with "Latest Posts" title and "See All" link
- Injects Person JSON-LD via `personJsonLd()` as inline script

### src/pages/about.astro
- Simple page wrapping HeroSection in BaseLayout
- Title: "About | Igor Zharikov Blog"

### src/pages/blogs/index.astro
- Fetches all 9 blog posts, sorts by date descending
- Passes all posts to BlogGrid with title "All Posts" and `showLink={false}`

### src/pages/blogs/[slug].astro
- `getStaticPaths` returns one route per post using `post.id` as slug param
- Calls `render(post)` from `astro:content` to get MDX Content component
- Wraps in BlogLayout passing all post metadata (title, description, date, slug, logo, tags)

### Deleted: src/pages/test-collections.astro
- Phase 1 smoke test page removed as planned

### Sitemap
- Auto-generated by `@astrojs/sitemap` integration already present in astro.config.mjs
- Accessible at `/sitemap-index.xml` — no additional code needed
