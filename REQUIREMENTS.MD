## Project Requirements

### 1. Technologies
- **Framework:** Next.js (React, shadcn/ui)
- **Hosting:** Cloudflare Pages
- **Content Format:** Markdown/MDX

### 2. Core Sections
- **Home**
- **About Me**
- **Blogs**

### 3. Global Layout
- **Header:** Navigation (Home, About Me, Blogs)
- **Footer:** Social media links
